# Milton – Tesis UMPE
Autor: Gabriel López del Cerro

Este repositorio contiene la documentación y el código utilizado en la construcción de los flujos de n8n del asistente financiero Milton. Dado que el entorno n8n pertenece a AstroPay y no puede ser compartido directamente, aquí se reproducen todos los nodos, funciones, prompts, esquemas y explicaciones necesarias para evaluar el trabajo.

## Estructura del repositorio

- `/docs/` — Explicación de cada flujo y cada nodo.
- `/src/` — Código JavaScript de todos los Function nodes.
- `/imagenes/` — Capturas de pantallas de flujos y nodos.

Más adelante completaré detalle de cada flujo.
